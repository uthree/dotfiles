# dotfiles

## Requirements
- Zsh
- Vim
- git

## Screenshot
![Screenshot](https://twitter.com/UThree271828/status/1524575005316907009/photo/1)


## Installation

1. Clone this repository
```sh
git clone https://github.com/uthree/dotfiles
```

2. Install
```sh
cd dotfiles
sh auto-install.sh
```

3. Install Vim plugins (vim required)
```sh
vim # Open vim
```
```
:PlugInstall
:PlugUpdate 
```

4. Install language server (optional)
```
:LspInstall
```
